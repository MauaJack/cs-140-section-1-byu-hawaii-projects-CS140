Web

How are the internet and web different?

The Internet is the system that connects all computers, while the Web is just one part of the Internet, made up of websites and webpages that you can visit with a browser.

ASCII and Unicode

ASCII is an old system for representing text on a computer, using just 128 characters. Unicode is a newer system that includes characters from almost all languages and symbols, making it more versatile.

What is UFT-8

UTF-8 is a way of encoding characters so that a computer can understand and display text. It can handle all characters in Unicode, including symbols and letters from different languages.

What is UTF-8

URL

A URL is the web address you type into your browser to visit a website. For example, "www.google.com" is a URL.

Domain Name

A domain name is a human-friendly address for a website.

HTTP

HTTP is the method used to transfer information between your computer and a website.

HTTPS

HTTPS is a secure version of HTTP. It encrypts the data between your browser and a website, making sure your personal information stays private and safe from hackers.

IP Address

An IP address is like a home address for your computer or phone on the Internet.

DNS

DNS works like an Internet phonebook.

W3C Recommendation

W3C Recommendations are a set of rules created to make websites work well across different devices and browsers.

XML, XHTML, HTML, HTML5

XML is a way to store and share data in a simple, organized format. XHTML is a stricter version of HTML that makes sure the code is cleaner and more organized. HTML is the basic code used to create webpages. HTML5 is the newest version of HTML, with new features like support for videos and better ways to structure web pages.
