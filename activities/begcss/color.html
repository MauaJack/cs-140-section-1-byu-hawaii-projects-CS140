  2. What is the difference between an ID and a Class?

    An ID is unique and can be used only once per page to identify a specific element, while a Class can be reused on multiple elements to apply the same styles.

  3. What is the difference between a SPAN tag and a DIV tag?

    A SPAN tag is an inline element used for styling small portions of content, while a DIV tag is a block-level element used to group larger sections of content or layout.

  4. What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

    A DIV tag is a generic container with no semantic meaning, while HEADER, MAIN, and FOOTER are semantic tags that define specific sections of a page, such as the header, main content, and footer, respectively.

    5. What are the different ways to select a color?

    The different ways to select a color in CSS are by using named colors, hexadecimal values, RGB values, RGBA values (with transparency), and HSL values (hue, saturation, lightness).
