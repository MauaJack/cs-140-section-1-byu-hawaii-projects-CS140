Answer these questions on your page:

  1. What is JavaScript? How did it get its name?

    JavaScript is a versatile, high-level programming language used primarily for web development to create interactive and dynamic web pages. It got its name as a marketing strategy by Netscape to leverage the popularity of Java, even though JavaScript and Java are fundamentally different languages.

  2. What are the two ways you can run JavaScript?

    JavaScript can be run in a web browser using its built-in JavaScript engine or on a server using a runtime environment like Node.js.

  3. What was the most popular front-end js library?

    jQuery was the most popular front-end JavaScript library for many years due to its ease of use and cross-browser compatibility.

  4. What is the most popular front-end js library today?

    Today, React.js is the most popular front-end JavaScript library, widely used for building dynamic and scalable web applications.
