  1. What is AJAX?

    AJAX (Asynchronous JavaScript and XML) is a web development technique that allows for the asynchronous loading of data from a server without refreshing the entire web page, enabling more dynamic and responsive user interfaces.

  2. Why do you put your JS Code in a separate file?

    Putting JavaScript code in a separate file promotes better organization, reusability, and maintainability of code, while also improving page load times through caching.
