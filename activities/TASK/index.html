Graphics

  1. What kind of photos are good to encode using JPEG?

    JPEG is good for photos like portraits and landscapes because it reduces file size while keeping most details.

  2. What kind of photos are good to encode using GIF?

    GIF is best for simple graphics or animations with fewer colors, like icons or logos.

  3. What is unique about the PNG format?

    PNG is unique because it has lossless compression and supports transparent backgrounds

  4. How do you resize your photo to the size it should be displayed?

    To resize a photo, use editing software to adjust its dimensions to match the display size you need.

  5. Why should you resize photos using photo editing software instead of resizing it using CSS?

    Resizing photos with editing software is better because it reduces file size and maintains quality, while CSS only changes the display size, not the actual file.
