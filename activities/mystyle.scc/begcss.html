  1. Explain the difference between inline, embedded, and external CSS.

    Inline CSS is applied directly to an HTML element, embedded CSS is written within a style tag in the document's head, and external CSS is stored in a separate file linked to the html.

  2. Explain why you should only use external External CSS should be used because it keeps HTML clean, promotes reusability across multiple pages, and improves website performance by allowing styles to be cached
  3. What is a selector used for?

    A selector in CSS is used to target specific HTML elements in order to apply styles to them.
