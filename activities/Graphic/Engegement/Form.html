  1. The form tag has an "action" attribute. What does it do?

    The action attribute in a

    tag specifies the URL where the form's data will be sent when submitted.

  2. The form tag has a "method" attribute. What does it do?

    The method attribute in a tag specifies the HTTP method (GET or POST) used to send the form's data to the server.

  3. What is the difference between a "POST" and a "GET"?

    The main differences between POST and GET are: GET: Sends data via the URL query string, making it visible in the address bar. It's best for retrieving data and should not be used for sensitive information. POST: Sends data in the request body, keeping it hidden from the URL. It's more secure and used for submitting forms, uploading files, and modifying data on the server.
